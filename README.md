# XACP V1 Preview

Experimental ARM coprocessor framework for the ZZ9000.

## Current validated features

- ARM MP3 decoding
- MP3 streaming pipeline
- Core1 dynamic launch
- Mandelbrot / Julia rendering
- Shared DDR mailbox
- XACP opcode protocol
- Experimental ARM compute offload

- firmware/
- docs/

## Status

Experimental developer preview.
Not intended for end users yet.
