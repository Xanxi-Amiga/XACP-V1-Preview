# XACP-V1
eXtended ARM coprocessor protocol for the ZZ9000 Amiga RTG board.

## Current validated features

- ARM MP3 decoding
- MP3 streaming pipeline
- Core1 dynamic launch
- Mandelbrot / Julia rendering
- Shared DDR mailbox
- XACP opcode protocol
- Experimental ARM compute offload

## Repository contents

- firmware/
- docs/
- sdk/
- applications/

## Status

Public beta firmware for ARM-assisted multimedia features on ZZ9000.
Includes working MP3 playback, GUI tools and still Scanlines V2 enhancements.
Tested on real Amiga hardware.
