# XACP Core1 Julia V2

This demo represents a newer generation of XACP rendering experiments using Core1 runtime execution on the ZZ9000 ARM subsystem.

Unlike the earlier Mandelbrot demonstrations, this version remains functional on recent XX16c firmware revisions.

Validated features:

* Core1 runtime execution
* Real-time Julia rendering
* Direct RTG framebuffer rendering
* Shared DDR communication
* ARM compute offloading
* Continuous rendering loop

This demo demonstrates that the XACP architecture evolved beyond the original Mandelbrot experiments into a more advanced multi-core ARM execution model.
